SmartChef IA Project

Developed in 2025

General Description

SmartChef IA is an artificial intelligence-based culinary assistant designed to reduce food waste. The project allows users to recognize ingredients using a camera and suggests personalized recipes instantly.

Smart Features

  • Visual Ingredient Recognition: It uses a computer vision model (TensorFlow.js) to identify fruits, vegetables, and proteins simply by showing them to the mobile camera.
  • Personalized Recipe Generator: Through the GPT-4o API, the system generates recipes based strictly on the detected ingredients, adapting to dietary preferences.
  • Nutritional Calculator: A JavaScript algorithm that calculates the approximate energy value of the final dish based on the identified ingredient database.
